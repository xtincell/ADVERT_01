## ðŸŽ¨ DESIGN STANDARDS (All Documents)

### Typography Hierarchy

**Headings:**
- H1 (Document Title): 32-36pt, Bold, Serif
- H2 (Section): 24-28pt, Bold, Serif
- H3 (Subsection): 18-20pt, Semibold, Serif
- H4 (Sub-subsection): 16pt, Semibold, Sans-serif
- H5 (Minor heading): 14pt, Medium, Sans-serif

**Body Text:**
- Regular: 11-12pt, Regular, Sans-serif, 1.5 line height
- Captions: 9-10pt, Regular, Sans-serif
- Quotes: 13-14pt, Italic, Serif

**Recommended Fonts:**
- Serif: Playfair Display, Lora, Merriweather
- Sans-serif: Inter, Helvetica Neue, Avenir

### Color Usage

**Primary Palette:**
- Brand Primary: Defined by client brand
- Brand Secondary: Defined by client brand
- Neutral Dark: #1F2937 (text)
- Neutral Mid: #6B7280 (secondary text)
- Neutral Light: #F3F4F6 (backgrounds)

**Semantic Colors:**
- Success: #10B981 (green)
- Warning: #F59E0B (amber)
- Danger: #EF4444 (red)
- Info: #3B82F6 (blue)

### Layout Standards

**Margins:**
- Top/Bottom: 1.5-2 inches
- Left/Right: 1.25-1.5 inches

**Spacing:**
- Section spacing: 24-32pt
- Paragraph spacing: 12-16pt
- List item spacing: 8pt

**Grid:**
- 12-column grid system
- Gutter: 20-24px

## ðŸ“Š DATA STANDARDS

### Variable Naming

Format: `pillar_category_field`

Examples:
- `a_purpose_statement`
- `d_color_primary_hex`
- `e_campaign_big_idea`
- `t_metric_north_star`

### Data Types

- **Text Short:** <500 characters (e.g., tagline, title)
- **Text Long:** <5000 characters (e.g., description, story)
- **Number:** Integer or decimal
- **Currency:** Number with currency code (USD, EUR, etc.)
- **Date:** ISO 8601 format (YYYY-MM-DD)
- **List:** Array of items
- **Object:** Nested structure
- **File:** Reference to asset (image, PDF, etc.)

### Required vs Optional

**Required Fields** (marked with *):
- Must be filled for document to be complete
- Blocks progression if missing

**Optional Fields**:
- Enhances document but not critical
- Can be added in future iterations
