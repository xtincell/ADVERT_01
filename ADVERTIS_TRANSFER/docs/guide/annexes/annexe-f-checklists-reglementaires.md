## Mode d'emploi

Ces checklists sont des **aide-mémoire**, pas des avis juridiques. Elles vous aident à identifier les zones de risque réglementaire pour chaque marché avant de finaliser la stratégie de marque avec votre client.

**Important :**
- Ces informations sont à jour au moment de la rédaction (février 2026) — vérifiez toujours les évolutions récentes
- Recommandez systématiquement un avocat spécialisé pour la validation finale
- 36 des 55 États de l'Union Africaine ont adopté des lois de protection des données (65% au 01/2024)
- Il n'existe PAS de standard publicitaire panafricain unifié

**Légende :** ☐ = À vérifier | ✓ = Conforme | ✗ = Non-conforme | N/A = Non applicable

## Checklist 1 : Afrique du Sud

### Cadre Général

Le marché sud-africain est le plus structuré du continent en matière de régulation publicitaire et de protection des données.

### A. Publicité et Communication

**Organisme de régulation :** Advertising Regulatory Board (ARB) — Code basé sur le Code International des Pratiques Publicitaires (ICC)

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Les claims publicitaires sont vérifiables et documentés | | |
| 2 | ☐ Les témoignages sont authentiques et autorisés par écrit | | |
| 3 | ☐ Les comparaisons concurrentielles sont factuelles et équitables | | |
| 4 | ☐ Le contenu ne cible pas les enfants de manière inappropriée | | |
| 5 | ☐ Les publicités alcool/tabac respectent les restrictions sectorielles | | |
| 6 | ☐ Le marketing d'influence divulgue les partenariats payés (#ad, #sponsored) | | |
| 7 | ☐ Les promotions et concours respectent le Consumer Protection Act (CPA) | | |
| 8 | ☐ Les prix affichés incluent la TVA (15%) | | |

### B. Protection des Données — POPIA

**Loi :** Protection of Personal Information Act (POPIA) — alignée avec le RGPD européen

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Consentement explicite obtenu AVANT collecte de données personnelles | | |
| 2 | ☐ Finalité de la collecte clairement communiquée au sujet des données | | |
| 3 | ☐ Un Information Officer est désigné et enregistré auprès du régulateur | | |
| 4 | ☐ Les données sont stockées dans un environnement sécurisé | | |
| 5 | ☐ Le droit d'accès, de correction et de suppression est respecté | | |
| 6 | ☐ Les transferts transfrontaliers de données respectent les conditions POPIA | | |
| 7 | ☐ Un PAIA Manual (Promotion of Access to Information Act) est publié | | |
| 8 | ☐ Procédure de notification de violation de données en place (72h) | | |

### C. Propriété Intellectuelle

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Marque déposée auprès de la CIPC (Companies and Intellectual Property Commission) | | |
| 2 | ☐ Recherche d'antériorité effectuée avant le dépôt | | |
| 3 | ☐ Design et logos protégés par le Designs Act | | |
| 4 | ☐ Contenu créatif protégé par le Copyright Act (pas d'enregistrement requis) | | |
| 5 | ☐ Clauses de PI incluses dans les contrats avec prestataires créatifs | | |

### D. Standards Sectoriels Spécifiques

| Secteur | Régulateur additionnel | Point clé |
|---|---|---|
| Finance | FSCA (Financial Sector Conduct Authority) | Obligations de divulgation renforcées |
| Santé | SAHPRA | Interdiction de claims médicaux non prouvés |
| Alimentation | NRCS | Étiquetage nutritionnel obligatoire |
| Télécommunications | ICASA | Règles sur le marketing direct et SMS |

## Checklist 2 : Nigeria

### Cadre Général

Plus grand marché du continent par la population. Réglementation en développement rapide, notamment sur le digital.

### A. Publicité et Communication

**Organisme de régulation :** Advertising Regulatory Council of Nigeria (ARCON)

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Toute publicité est approuvée par ARCON avant diffusion | | |
| 2 | ☐ Les praticiens publicitaires sont enregistrés auprès d'ARCON | | |
| 3 | ☐ Les publicités extérieures (billboards) ont un permis ARCON valide | | |
| 4 | ☐ Le contenu ne contient pas de claims trompeurs ou exagérés | | |
| 5 | ☐ Les publicités de produits réglementés (alcool, pharma) ont l'approbation sectorielle | | |
| 6 | ☐ Le marketing d'influence respecte les guidelines émergentes d'ARCON sur le digital | | |
| 7 | ☐ Les prix sont en Naira (₦) pour les publicités ciblant le marché local | | |

### B. Protection des Données — NDPR / NDPA

**Loi :** Nigeria Data Protection Regulation (NDPR) → remplacée par le Nigeria Data Protection Act (NDPA, 2023)

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Consentement explicite obtenu avant traitement des données personnelles | | |
| 2 | ☐ Politique de confidentialité publiée et accessible | | |
| 3 | ☐ Un Data Protection Officer (DPO) est désigné | | |
| 4 | ☐ Audit annuel de protection des données effectué (obligatoire si >200 sujets) | | |
| 5 | ☐ Les données sensibles bénéficient de protections renforcées | | |
| 6 | ☐ Le transfert international de données est encadré | | |
| 7 | ☐ Le droit à l'oubli et à la portabilité est respecté | | |

### C. Propriété Intellectuelle

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Marque déposée auprès du Trademarks, Patents and Designs Registry | | |
| 2 | ☐ Le dépôt couvre les classes pertinentes (Nice Classification) | | |
| 3 | ☐ Protection du copyright enregistrée auprès de la Nigerian Copyright Commission | | |
| 4 | ☐ Contrats de cession de PI signés avec les prestataires créatifs | | |

## Checklist 3 : Kenya

### Cadre Général

Hub technologique d'Afrique de l'Est. Écosystème startup dynamique avec une réglementation relativement avancée.

### A. Publicité et Communication

**Organisme :** Advertising Standards Board of Kenya (ASBK) — autorégulation

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Les publicités sont honnêtes, véridiques et décentes | | |
| 2 | ☐ Les comparaisons sont objectives et vérifiables | | |
| 3 | ☐ Les publicités ciblant les enfants respectent les restrictions | | |
| 4 | ☐ Les promotions respectent le Competition Act | | |
| 5 | ☐ Le contenu SMS/mobile marketing est opt-in | | |
| 6 | ☐ Les publicités online respectent les guidelines du Kenya ICT Authority | | |

### B. Protection des Données — DPA

| # | Point de contrôle | Statut | Notes |

| 2 | ☐ Consentement informé et spécifique obtenu | | |
| 3 | ☐ Data Protection Impact Assessment (DPIA) effectué pour les traitements à risque | | |
| 4 | ☐ Notification de violation dans les 72 heures | | |
| 5 | ☐ Politique de rétention des données définie | | |
| 6 | ☐ Transferts transfrontaliers conformes aux conditions du DPA | | |

### C. Propriété Intellectuelle

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Marque déposée auprès de la Kenya Industrial Property Institute (KIPI) | | |
| 2 | ☐ Recherche d'antériorité effectuée | | |

## Checklist 4 : Zone UEMOA / CEMAC (Afrique Francophone)

### Cadre Général

Deux unions économiques couvrant 22 pays. Réglementation harmonisée à l'échelle régionale avec des spécificités nationales. L'ARPP (France) sert de référence d'autorégulation pour les marchés francophones.

### A. Publicité et Communication

**Cadre :** ARPP comme référence + législations nationales

| # | Point de contrôle | Statut | Notes / Pays |
|---|---|---|---|
| 1 | ☐ Les publicités ne contiennent pas de claims trompeurs (loi sur la concurrence) | | |
| 2 | ☐ Les publicités comparatives respectent les conditions légales du pays cible | | |
| 3 | ☐ Les publicités en langues locales respectent les normes culturelles | | |
| 4 | ☐ Les publicités de produits importés mentionnent le pays d'origine | | |
| 5 | ☐ Les promotions sont déclarées auprès des autorités compétentes (Côte d'Ivoire, Sénégal) | | |
| 6 | ☐ Les publicités digitales respectent les CGU des plateformes ET les lois locales | | |
| 7 | ☐ Les influenceurs déclarent les partenariats commerciaux | | |

### B. Protection des Données

**Cadre :** Convention de l'Union Africaine sur la cybersécurité et la protection des données personnelles (Malabo, 2014) + lois nationales

| Pays | Loi | Autorité | Statut |
|---|---|---|---|
| Sénégal | Loi 2008-12 | Commission de Données Personnelles (CDP) | Active |
| Côte d'Ivoire | Loi 2013-450 | ARTCI | Active |
| Burkina Faso | Loi 2021-031 | Commission Informatique et Libertés (CIL) | Active |
| Mali | Loi 2013-015 | APDP | Active |
| Cameroun | Loi 2024 | — | Récente |
| Gabon | Loi 2011 | CNPDCP | Active |

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Vérifier quelle loi s'applique dans le(s) pays cible(s) | | |
| 2 | ☐ Consentement préalable obtenu (souvent exigé par les lois francophones) | | |
| 3 | ☐ Déclaration ou autorisation auprès de l'autorité compétente | | |
| 4 | ☐ Données hébergées conformément aux exigences de localisation (si applicable) | | |
| 5 | ☐ Le droit d'accès, de rectification et d'opposition est garanti | | |

### C. Propriété Intellectuelle — OAPI

**Organisme :** Organisation Africaine de la Propriété Intellectuelle (OAPI) — 17 États membres

| # | Point de contrôle | Statut | Notes |
|---|---|---|---|
| 1 | ☐ Marque déposée auprès de l'OAPI (un seul dépôt couvre 17 pays) | | |
| 2 | ☐ Recherche d'antériorité via la base OAPI | | |
| 3 | ☐ Renouvellement prévu (10 ans) | | |
| 4 | ☐ Protection cumulative par le droit d'auteur (Accord de Bangui) | | |

## Checklist 5 : Maroc / Tunisie (Afrique du Nord)

### Cadre Général

Marchés structurés avec une forte influence du cadre réglementaire européen, notamment français.

### A. Publicité et Communication

**Maroc :** Loi 77-03 sur la communication audiovisuelle + HACA (Haute Autorité de la Communication Audiovisuelle)
**Tunisie :** Décret-loi 2011-116 sur la liberté de la presse + HAICA

| # | Point de contrôle | Statut | Pays |
|---|---|---|---|
| 1 | ☐ Les publicités respectent les « bonnes mœurs » et l'ordre public | | MAR / TUN |
| 2 | ☐ Les publicités en arabe et en français respectent les normes linguistiques | | MAR / TUN |
| 3 | ☐ Les publicités de produits réglementés (alcool, tabac, pharma) sont conformes | | MAR / TUN |
| 4 | ☐ Les publicités digitales transfrontalières respectent les lois du pays de diffusion | | MAR / TUN |
| 5 | ☐ Les concours et loteries ont une autorisation préfectorale/gouvernorale | | MAR / TUN |
| 6 | ☐ Le contenu respecte les sensibilités religieuses et culturelles locales | | MAR / TUN |

### B. Protection des Données

**Maroc :** Loi 09-08 — supervisée par la CNDP (Commission Nationale de contrôle de la protection des Données à caractère Personnel)
**Tunisie :** Loi organique 2004-63 — supervisée par l'INPDP

| # | Point de contrôle | Statut | Pays |
|---|---|---|---|
| 1 | ☐ Déclaration préalable auprès de la CNDP / INPDP | | MAR / TUN |
| 2 | ☐ Consentement éclairé et préalable obtenu | | MAR / TUN |
| 3 | ☐ Finalité du traitement clairement définie et respectée | | MAR / TUN |
| 4 | ☐ Durée de conservation des données définie | | MAR / TUN |
| 5 | ☐ Transfert international encadré (autorisation CNDP requise au Maroc) | | MAR |
| 6 | ☐ Droit d'accès, de rectification et d'opposition garanti | | MAR / TUN |
| 7 | ☐ Mesures de sécurité techniques et organisationnelles en place | | MAR / TUN |

### C. Propriété Intellectuelle

**Maroc :** OMPIC (Office Marocain de la Propriété Industrielle et Commerciale)
**Tunisie :** INNORPI (Institut National de la Normalisation et de la Propriété Industrielle)

| # | Point de contrôle | Statut | Pays |
|---|---|---|---|
| 1 | ☐ Marque déposée auprès de l'OMPIC / INNORPI | | MAR / TUN |
| 2 | ☐ Recherche d'antériorité nationale effectuée | | MAR / TUN |
| 3 | ☐ Extension Madrid possible pour protection internationale | | MAR / TUN |
| 4 | ☐ Contrats créatifs incluent des clauses de cession de droits d'auteur | | MAR / TUN |

## Matrice de Synthèse Rapide

| Dimension | Afrique du Sud | Nigeria | Kenya | UEMOA/CEMAC | Maroc/Tunisie |
|---|---|---|---|---|---|
| **Maturité réglementaire** | Élevée | Moyenne | Moyenne | Variable | Élevée |
| **Protection données** | POPIA (fort) | NDPA (récent) | DPA (moyen) | Variable | Fort |
| **Régulation pub** | ARB (auto) | ARCON (obligatoire) | ASBK (auto) | Variable | Mixte |
| **PI — Dépôt marque** | CIPC | Registry | KIPI | OAPI (régional) | OMPIC / INNORPI |
| **Risque principal** | POPIA sanctions | Pré-approbation ARCON | Évolution rapide | Hétérogénéité | Sensibilité culturelle |

---

**Rappel :** Ces checklists sont un point de départ. Pour chaque nouveau marché, prévoyez un budget « due diligence réglementaire » dans votre offre de services. Un problème de conformité peut détruire une marque plus vite qu'une mauvaise stratégie.
